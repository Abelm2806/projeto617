Sobre mim

O meu nome é José Brito e estou a desenvolver uma página pessoal em HTML.

Estou neste curso para aprender melhor como funcionam a área da programação.

Quero aprender a escrever código e a usar ferramentas como Git e GitHub.

Percurso

Tenho vindo a aprender conceitos básicos de html e a perceber melhor como uma página web é estruturada.

Estou a aprender a usar o git e o github para gerir os meus projetos de forma mais eficiente.

Uma das partes mais desafiantes tem sido trabalhar com commits, branches e pull requests, mas também tem sido muito gratificante ver o meu progresso e a minha evolução.

O que estou a aprender

Nesta unidade estou a praticar várias etapas importantes para trabalhar com o git e o GitHub.

  • Criar branches para trabalhar numa tarefa sem alterar diretamente o main
  • Fazer commits pequenos com mensagens claras
  • Fazer pull requests para pedir revisão do trabalho

Ferramentas usadas

Durante este projeto usei algumas ferramentas para escrever código, testar a página e guardar as alterações.

  • VS Code para escrever o ficheiro html
  • Git para controlar as versões do projeto
  • GitHub para enviar o branch e abrir o pull request

Comandos praticados

Ao longo da tarefa pratiquei comandos e ações uteis para trabalhar com o Git.

  • git status para verificar alterações
  • git add e git commit para guardar as alterações
  • git push para enviar o branch para o GitHub e abrir o pull request
